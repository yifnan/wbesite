What does the project do?

Our goal for this project is to be able to create a model that can identify whether or not a person has diabetes. By using a dataset that includes several health levels such as: Glucose, Insulin, and Age, our team is able to train our model to identify the symptom levels of diabetes.

Machine Learning Models





Logistic Regression

Logistic Regression is used when predicting an outcome based on a fixed variable such as determining possible risk of disease using health factors like Glucose level and BMI (Body Mass Index). Before predicting an outcome, filtering through data is usually the best course of action because it allows faulty data to be discarded such as if the data set displays someone’s blood pressure as 0 or 400.

Logistic Regression accuracy - 84.6 %

K-nearest neighbors

KNN takes a data point and find the k nearest points to that data point. If k=5, KNN find the 5 nearest neighbors and chooses the class of neighbor that has the majority. This is why k should not be an even number because there is a possibility of a tie. In this dataset I tested many different k values and found that k=47 gave the best accuracy.

K-nearest neighbors accuracy - 85.9 %

Random Forests

Random forest works by constructing a “forest” of decision trees and outputting the average result from the trees to categorize data. To increase the accuracy of the method, we can go through the data set and take out outliers and only apply the most relevant data points.





-Conclusion-

Based on the results, our team has come to the conclusion that k-nearest neighbors has performed the best compared to all the ML models. With cleaning the data and handling the outliers, all models were able to get to significantly high accuracy points:

Logistic Regression accuracy - 84.6 %

K-nearest neighbors accuracy - 85.9 %

Random Forest accuracy - 80.7 %

Since our data set is not the most significant(big) and recent set, further research must be done in order to use this model in the real world. However with more recent research and data this model could be able to prevent people from experiencing harsh effects of untreated diabetes, along with helping people try and reverse it. If our model is able to point out diabetes early enough, then the individual has a chance of reversing their condition.
